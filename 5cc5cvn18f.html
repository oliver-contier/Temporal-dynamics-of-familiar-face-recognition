Code and data availability

Un-thresholded statistical maps are available on neurovault.org \cite{Gorgolewski_2015} under http://neurovault.org/collections/2843/. All analysis scripts are available at the following GitHub repository: https://github.com/oliver-contier/famface-temporal-dynamics. All data is available at http://datasets.datalad.org/?dir=/labs/gobbini/famface/data.
